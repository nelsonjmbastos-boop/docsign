# DocSign — Tecnovia Angola

Assinatura Digital de PDFs via dispositivo móvel.

## Deploy no GitHub Pages

1. Faz upload de todos os ficheiros para um repositório GitHub
2. Vai a Settings → Pages → Source: main branch
3. O link será: `https://[teu-username].github.io/docsign`

## Ficheiros
- `index.html` — App principal
- `manifest.json` — PWA manifest
- `sw.js` — Service worker (offline)
- `icon-*.png` — Ícones PWA
